# ZenFlow — Student Wellness AI

ZenFlow is a premium student wellness platform powered by AI. It features mood tracking, personalized wellness recommendations, focus sessions, and a holistic dashboard to help students manage their mental health and productivity.

## Features

- **AI Recommendations**: Personalized wellness suggestions based on your data.
- **Mood Tracking**: Log and visualize your emotional well-being over time.
- **Focus Sessions**: Smart scheduling and focus timers to boost productivity.
- **Night Mode**: Beautifully crafted dark mode for a better evening experience.
- **Responsive Design**: Works seamlessly across mobile and desktop.

## Deployment

The project is a single-page application (SPA) contained within `index.html`. It can be easily deployed to GitHub Pages or any static hosting service.
